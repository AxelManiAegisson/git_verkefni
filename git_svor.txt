#Git spurningar og svör

1. Hvað gera eftirfarandi Linux skipanir?
	cd
	cd ~
	ls
	pwd
	mkdir
Svar: 
	cd = Fer á annan stað
	cd ~ = Fer á rótinna
	ls = Gefur lista yfir efni í möppu eða directory
	pwd = Sýnir hvar þú ert staddur
	mkdir = Býr til nýtt directory
2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone
	git log
	git status
	git diff
	git checkout
Svar: 
	git clone = Klónar geymslu
	git log = Sýnir hvað þú ert búinn að gera
	git status = Sýnir statusinn á directoryinu
	git diff = Sýnir breytingar á milli commits og working tréð
	git checkout = Fara á milli branches	
3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff 
	b) git diff --staged 
	c) git diff commit1 commit2

4. 	Hvað er útgáfustýring (e.version control)?
Svar: 
	Það sér um breytingar á Skjölum, forritum, stórum síðum og öðrum stórum söfnum af efnum.
5.	Hverjir eru helstu kostir við að nota GIT?
Svar:
	Branches er góður kostur útaf því að þú getur byrjað að vinna í einhverju án þess að rústa master branchinu.
6. 	Hversu oft telur þú að eigi að gera commit í verkefni, rökstuddu?
Svar: 
	Alltaf þegar einhvað virkar og allir eru sammála
